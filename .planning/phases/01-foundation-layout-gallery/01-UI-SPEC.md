## Layout & Section Structure

### Section Order (top to bottom)

1. **NavBar** — Fixed/sticky top, smooth-scroll links to each section
2. **Hero** — Full-width, centered display heading, subtitle, "View Our Gallery" CTA
3. **About** — Text content, centered or split layout
4. **Gallery** — Responsive grid (2 cols mobile, 3 cols tablet, 4 cols desktop), lightbox on tap
5. **Videos** — Thumbnail grid (1 col mobile, 2 cols desktop), play overlay → inline embed
6. **Form Placeholder** — Centered card stating "Registration form coming soon"
7. **Footer** — Brand, social links, copyright

### Breakpoints

| Breakpoint | Width | Column Count (Gallery) | Column Count (Videos) |
|------------|-------|------------------------|------------------------|
| Mobile | < 640px | 2 | 1 |
| Tablet | 640px–1023px | 3 | 2 |
| Desktop | ≥ 1024px | 4 | 2 |

### Smooth Scroll

- Tap a nav link → `scroll-behavior: smooth` to scroll to `id` target
- Nav active state updates based on scroll position (IntersectionObserver)
- Offset scroll target by nav height (~64px)

## Responsive & Accessibility Requirements

### Mobile Adaptations
- Nav: Hamburger menu or condensed link list (stacked, full-width) on < 640px
- Touch targets: minimum 44×44px for all interactive elements (nav links, buttons, play overlays)
- Lightbox: swipe to navigate photos (touch events)
- Gallery grid: 2 columns on mobile
- Hero title: scales from 40px → 32px on mobile

### Accessibility
- All images must have descriptive `alt` text (dynamic alt from Google Drive metadata or fallback "Supper club event photo")
- Lightbox: focus trap within dialog, Escape to close
- Keyboard navigation: Tab through nav links, Enter to activate, arrow keys in lightbox
- Color contrast: verify #A16207 on #FEF2F2 passes WCAG AA for large text (CTA buttons). Body text #450A0A on #FEF2F2 passes AA.
- Smooth scroll respects `prefers-reduced-motion` (`@media (prefers-reduced-motion: reduce) { scroll-behavior: auto; }`)
